# Datos del proyecto

Estructura de `data/`:

| Carpeta | Contenido |
|---------|-----------|
| `raw/` | Datos descargados de internet (CSV, XLSX, TXT de fuentes externas) |
| `intermedio/` | Listas y artefactos generados para corridas (apellidos, pares, etc.) |
| `outputs/` | Resultados de evaluaciones y muestras (CSV, TXT de salida) |
| `db/` | Base SQLite `nombres.db` (generada por scripts, no editar a mano) |

Registro de URLs y fuentes: `raw/fuentes.txt`.
